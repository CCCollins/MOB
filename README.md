# Militech Open Bot

Автономный ИИ-агент (Оркестратор), способный взаимодействовать с системой пользователя, искать информацию в интернете и выполнять фоновые задачи через графический интерфейс или в скрытом (Headless) режиме.

## Структура проекта

* `run.py` — главная точка входа. Запускает GUI или Headless-режим.
* `config/` — настройки и сохранение конфигурации.
* `core/` — ядро: ИИ-агент, БД, инструменты ОС, фоновый runner.
* `interfaces/` — интерфейсы взаимодействия: Telegram-бот и GUI.

## Установка и запуск

1. **Linux**
   ```bash
   chmod +x install_linux.sh
   ./install_linux.sh
   ./run.sh
   ```

2. **Windows**
   ```bash
   pip install -r requirements.txt
   python run.py
   ```

## Headless-режим (без графики, для серверов)

Чтобы запустить агента на сервере без графического интерфейса (или в фоне):
```bash
python run.py --headless
```
В этом режиме бот будет доступен через Telegram, но инструменты графического интерфейса ОС (мышь, скриншоты) будут отключены, чтобы не вызвать падение.
